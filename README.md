# Stock Trading Platform

A Java console-based Stock Trading Platform developed for the CodeAlpha Java Internship.

## Features
- Display available stock details
- Buy shares
- Calculate total purchase amount
- Simple and user-friendly console interface
- Object-Oriented Programming (OOP) implementation

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac StockTradingPlatform.java

2. Run the program:
   java StockTradingPlatform

## Sample Output
=== Stock Trading Platform ===
Available Stock: TCS
Price: ₹3500.0
Enter number of shares to buy: 5

Purchase Successful!
Stock: TCS
Shares: 5
